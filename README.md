# Python CLI Calculator

A simple command-line calculator built in Python that supports basic arithmetic operations.

## Features

* Addition (+)
* Subtraction (-)
* Multiplication (*)
* Division (/)
* Handles division by zero
* Stores calculation history

## How It Works

The program takes user input in the format:

```
5 + 3
```

It then parses the input, performs the calculation, and displays the result.

## Example

```
Start Calculating: 5 + 3
5 + 3 = 8
```

## What I Learned

* Handling user input in Python
* Using `if/elif` for decision making
* Managing program flow with loops
* Preventing errors like division by zero
* Structuring cleaner and more readable code

## How to Run

1. Make sure Python is installed
2. Run the script:

```
python calculator.py
```
